AI情绪日记&心理状态智能陪伴系统 技术方案
一、方案核心定位
本方案打造一款以极致隐私安全为核心、AI 多模态情绪识别为驱动、专业心理陪伴为价值的移动端应用，核心解决用户情绪记录、心理疏导的核心需求，同时深度落地用户指定的自定义 APP 名称 / 图标、多层级密码保护、全链路隐私加密三大核心隐私诉求，严格遵循心理咨询伦理与个人信息保护合规要求，实现 “安全、私密、懂你、长效陪伴” 的产品目标。
二、整体技术架构
采用端侧优先、隐私原生的分层架构设计，核心计算与数据存储默认落地本地，云端服务仅作为可选扩展，从架构根源杜绝隐私泄露风险，整体架构分为 5 层：
架构层级	核心定位	核心能力
端侧应用层	用户交互与核心功能落地	日记录入、AI 陪伴、伪装功能、密码管控、可视化报告
端侧 AI 引擎层	离线 AI 能力核心	端侧情绪识别、语音转写、轻量大模型推理、多模态特征融合
隐私安全中台	全链路隐私管控	数据加密、权限管控、伪装适配、防泄露防护、密钥管理
可选云端服务层	扩展能力支撑（用户可一键关闭）	云端大模型服务、端到端加密备份、合规内容审核
合规与伦理管控层	全流程风险管控	心理咨询伦理对齐、内容安全、合规适配、极端风险预警
三、核心业务功能模块
3.1 多模态情绪日记录入模块
支持全场景、低门槛的情绪记录，所有录入内容默认即时端侧加密，不缓存明文：
1.富文本日记录入：支持文字、表情、标签、天气、时间戳、地点（可选）录入，支持情绪快捷标记，预设积极 / 中性 / 负面情绪标签库，支持用户自定义标签；
2.离线语音录入：基于端侧量化 Whisper-Tiny 模型，实现本地语音转文字，全程不上传音频数据，支持语音情绪同步识别；
3.图片 / 视频情绪录入：支持上传图片 / 短视频，端侧 CNN 模型本地识别面部表情、场景氛围，同步提取情绪特征，不上传原始媒体文件；
4.快捷情绪记录：预设一键记录模板，支持 10 秒快速标记当日情绪状态、触发事件，降低记录门槛。
3.2 AI 多模态情绪识别与分析模块
采用端侧 + 云端双轨可选架构，用户可自由选择纯离线模式，核心能力：
1.多维度情绪识别：基于中文心理健康数据集微调的 DistilBERT 模型，识别 8 种基础情绪（愉悦、愤怒、悲伤、恐惧、惊讶、厌恶、焦虑、孤独）+20 种复合情绪，输出情绪强度、情绪极性、核心触发词；
2.多模态融合分析：融合文字语义、语音声纹（语速、音调、音量）、图像表情三大维度特征，综合输出情绪判断，支持上下文关联分析，识别长期情绪模式；
3.长期情绪追踪与可视化：生成日 / 周 / 月 / 季度情绪趋势曲线、情绪热力图、触发词云、情绪波动归因分析，直观呈现用户心理状态变化；
4.个性化心理报告：定期生成心理健康状态报告，正向反馈用户积极行为，针对持续负面情绪给出温和的疏导建议，严格规避疾病诊断表述。
3.3 智能心理陪伴与干预模块
基于心理咨询伦理对齐的大模型微调实现，核心红线：不做心理疾病诊断、不推荐药物、不替代专业心理咨询，仅做共情陪伴与正向疏导：
1.共情式对话陪伴：7×24 小时响应，基于用户日记内容与情绪状态，生成贴合语境的共情回应，而非通用话术，支持开放式倾诉、情绪树洞、困惑解答；
2.分级情绪干预：针对不同情绪状态匹配对应干预方案，轻度负面情绪给出正念引导、呼吸训练、认知重构小练习；中度持续负面情绪给出 CBT 认知行为疗法分步练习；极端风险（自伤 / 自杀倾向）立即触发预警，推送全国心理援助热线，引导用户寻求专业帮助；
3.个性化成长方案：基于用户长期情绪模式，定制专属心理成长计划，如内耗缓解、社交自信提升、睡眠改善、压力管理等，支持每日打卡、进度追踪；
4.情绪急救工具箱：预设崩溃安抚、焦虑缓解、失眠助眠、愤怒平复等应急场景工具，一键启动对应干预内容，应对突发情绪危机。
3.4 数据可视化与成长激励模块
1.情绪数据全维度可视化，支持按时间、事件、标签筛选查看情绪变化；
2.正向成长激励体系，记录用户坚持记录、情绪调节、积极行为的成就，生成成长里程碑，强化积极心理建设；
3.支持数据一键导出（加密 / 明文可选），便于用户自行留存或在授权前提下提供给专业心理咨询师参考。
四、专项隐私安全功能模块（核心需求落地）
4.1 APP 自定义伪装功能（名称 + 图标全替换）
核心功能
实现桌面端 APP 完全隐身，彻底隐藏情绪日记的产品属性，避免他人察觉，支持全维度自定义：
1.桌面名称自定义：用户可自由修改 APP 在手机桌面显示的名称，无字符限制，预设常用伪装名称库（备忘录、计算器、天气、单词本、便签、学习笔记等），支持用户输入任意自定义名称；
2.桌面图标全自定义：
￮预设多套风格图标包（系统原生、简约办公、学习工具、生活工具等），覆盖主流伪装场景，一键切换；
￮支持用户自定义上传本地图片作为 APP 图标，自动适配不同手机桌面的图标尺寸规范；
3.深度伪装模式（计算器隐身）：可选开启，开启后 APP 桌面显示为计算器图标与名称，打开 APP 默认进入完全可用的标准计算器界面，仅在计算器中输入正确的密码 / 暗码，才能跳转进入真实的情绪日记系统，实现极致隐身；
4.场景化一键切换：支持预设多套伪装方案，一键切换，适配不同场景的隐私需求。
技术实现
•Android 端：采用activity-alias动态桌面入口技术，无需 ROOT 权限，通过启用 / 禁用不同别名组件，动态切换桌面图标与名称，兼容华为、小米、OPPO、vivo 等主流厂商定制系统，针对不支持动态名称的机型提供 fallback 适配方案；
•iOS 端：基于系统Alternate Icon官方能力实现图标切换，预设合规的备用图标包，名称通过本地化字符串动态适配，符合 App Store 审核规则；同时提供企业签名版，支持无限制自定义图标与名称；
•图标切换后无需重启手机，APP 重启后即可生效，切换过程无日志留存，无痕迹。
4.2 多层级密码保护功能
核心功能
全场景、多层级的访问管控，彻底杜绝未经授权的内容访问，覆盖用户所有隐私场景需求：
1.基础启动锁：APP 启动、从后台切换回前台时，必须验证身份才能进入，支持 4 种验证方式：数字密码、手势密码、指纹识别、面部生物识别；
2.双密码分级权限：核心防胁迫功能，设置两套独立密码：
￮主人密码：验证通过后进入真实的情绪日记系统，可使用全部功能；
￮访客 / 胁迫密码：验证通过后进入预设的伪装空白界面（如普通备忘录、空白计算器），完全隐藏真实日记内容，应对他人逼问、强行查看等场景；
3.紧急避险功能：
￮连续输错密码 N 次（用户可设置），自动触发前置摄像头抓拍（可选开启），同时自动清除当前界面，进入伪装模式；
￮摇一摇 / 一键电源键触发：手机摇一摇，或快速按两次电源键，立即切换到伪装界面，应对突发有人靠近的场景；
￮后台自动锁定：可设置无操作自动锁定时间（10 秒 / 30 秒 / 1 分钟 / 5 分钟可选），APP 退到后台立即锁定，防止忘记退出导致的泄露；
4.单条内容加密锁：支持对特定日记、对话记录设置单独的二级密码，即使进入主系统，也需要额外验证才能查看，保护极致私密内容；
5.生物识别安全管控：指纹 / 面部识别仅调用系统原生 API，生物数据全程在设备本地验证，APP 不采集、不上传任何生物特征信息，杜绝生物数据泄露。
技术实现
•密码采用 PBKDF2 算法加盐哈希存储，本地不保存明文密码，密钥由用户密码派生，无法逆向破解；
•生物识别调用 Android Biometric API、iOS LocalAuthentication 系统原生能力，全程在系统安全区（TEE）内完成验证，APP 无法获取生物数据；
•界面锁定采用 Activity 栈管控，锁定状态下拦截所有界面跳转，内存中不缓存敏感界面内容，防止截屏、录屏、内存 dump 泄露。
4.3 全链路隐私安全增强功能
1.端侧全加密存储：所有日记内容、音频、图片、对话记录，全部采用 AES-256 算法本地加密存储，密钥由用户主密码派生，只有用户本人可解密；数据库采用 SQLCipher 全库加密，防止 root / 越狱设备的数据提取；
2.纯离线模式开关：一键开启后，APP 完全断网，禁用所有网络权限，所有 AI 计算、情绪识别、日记存储、陪伴对话全部在设备本地完成，不产生任何网络流量，从根源上杜绝数据上传泄露；
3.端到端云端备份（可选）：用户可自主选择是否开启云端备份，备份内容在手机本地完成加密后再上传，云端仅存储密文，服务商无密钥无法解密，即使云端数据泄露，也无法获取用户明文内容；
4.数据可控全生命周期管理：
￮支持一键导出所有加密 / 明文数据，格式兼容 Excel、PDF、TXT；
￮支持一键销毁所有本地 + 云端数据，执行硬删除 + 数据覆写，不可恢复，无任何残留；
￮支持定时自动销毁，可设置日记在指定时间后自动永久删除；
5.防泄露防护体系：
￮敏感界面防截屏 / 录屏：日记详情、AI 对话等敏感界面，禁用系统截屏、录屏功能，Android 端通过FLAG_SECURE实现，iOS 端通过屏幕捕获拦截实现；
￮键盘隐私保护：敏感输入场景禁用第三方键盘，强制使用系统原生键盘，关闭键盘联想与输入缓存，防止第三方键盘记录用户日记内容；
￮防调试、防逆向：APP 加壳保护，防反编译、防调试、防内存注入，防止加密逻辑被破解；
6.合规隐私管控：
￮数据最小化原则：不强制用户注册账号，游客模式即可使用所有核心功能，不强制收集手机号、身份证等个人信息；
￮全流程授权管控：所有权限、数据采集、云端服务，均需用户明确主动授权，可随时撤回授权，删除对应数据；
￮符合《个人信息保护法》《GDPR》《中国心理学会临床与咨询伦理守则》，提供清晰透明的隐私政策，明确告知用户数据用途与边界。
五、核心技术选型
5.1 移动端开发选型
技术方向	选型方案	选型理由
主开发框架	优先 Flutter，备选原生开发（Android Kotlin + iOS Swift）	Flutter 一套代码双端适配，UI 自定义能力强，性能接近原生，适配图标 / 名称动态切换；原生开发可实现更精细的权限管控与隐私防护，适配极致安全需求
端侧 AI 框架	TensorFlow Lite、PyTorch Mobile、Google ML Kit	轻量化、高性能，支持端侧模型量化推理，适配离线 AI 能力落地
加密库	OpenSSL、BouncyCastle、SQLCipher	行业标准加密算法，支持 AES-256 全库加密、密钥安全派生，适配全平台
生物识别	系统原生 API（Android Biometric API /iOS LocalAuthentication）	全程本地验证，不采集生物数据，符合系统安全规范
5.2 AI 模型选型
能力场景	端侧离线模型	云端可选模型
文本情绪识别	基于 DistilBERT 微调的中文情感分析模型，量化后 < 100MB	字节豆包大模型、通义千问大模型（心理健康领域微调）
语音转写与情绪识别	Whisper-Tiny 量化版 + CNN-LSTM 轻量声纹情绪模型	阿里云语音识别、腾讯云语音情绪识别服务
图像情绪识别	MobileNet 轻量化面部表情识别模型	百度 AI 多模态情绪识别服务
智能陪伴对话	Llama 3 8B/Qwen 7B 量化版，基于中文心理健康对话数据集微调对齐	字节豆包大模型 4.0、通义千问 4.0（心理咨询伦理对齐 + 安全对齐）
5.3 服务端技术选型（可选扩展）
•后端框架：FastAPI（Python），高性能异步，适配 AI 接口对接；
•数据库：MySQL（用户基础信息，哈希加密存储）、MinIO/OSS（端到端加密备份数据存储）；
•部署架构：Docker+K8s 容器化部署，支持私有化部署，适配 B 端机构定制需求；
•接口安全：HTTPS 双向认证、API 签名鉴权、请求频率限制、防 DDoS 攻击。

七、伦理合规与风险管控
1.心理咨询伦理红线：AI 模型严格遵循《中国心理学会临床与咨询心理学工作伦理守则》，明确标注 “本工具仅为情绪陪伴工具，不能替代专业心理咨询、心理治疗与医疗诊断”，严禁输出心理疾病诊断、药物推荐等违规内容；
2.极端风险预警：针对自伤、自杀、暴力等极端风险内容，建立分级预警机制，轻度风险给出正向引导，重度风险立即推送全国心理援助热线，引导用户寻求专业机构帮助，必要时在合规前提下触发紧急联系人通知（用户提前授权）；
3.内容安全管控：建立全流程内容安全审核机制，AI 模型提前做有害内容对齐，杜绝输出歧视、诱导、伤害性内容，保障用户心理健康安全；
4.免责声明：在 APP 首次启动、隐私政策、设置页面，明确告知用户工具边界与风险，提示用户若存在严重心理困扰，及时寻求专业医疗机构与心理咨询师的帮助。
