# 下载页面设计说明

## 🎨 设计理念

参考QQ和豆包的下载页面设计，打造现代化、专业化的应用下载页面。

### 设计特点

#### 1. 响应式设计
- ✅ 完美适配手机、平板、电脑
- ✅ 流式布局，自动适应屏幕尺寸
- ✅ 移动优先设计理念

#### 2. 暖橙色主题
- ✅ 与应用风格保持一致
- ✅ 使用渐变色增强视觉效果
- ✅ 温暖友好的品牌形象

#### 3. 现代化UI元素
- ✅ 圆角卡片设计
- ✅ 柔和阴影效果
- ✅ 平滑动画过渡
- ✅ 玻璃态导航栏

## 🎨 配色方案

### 颜色变量
```css
:root {
    /* 主色调 */
    --primary-color: #EA580C;      /* 暖橙色 */
    --primary-light: #FFF7ED;      /* 浅橙色背景 */
    --primary-dark: #C2410C;       /* 深橙色 */
    --secondary-color: #F97316;    /* 辅助橙色 */

    /* 文字颜色 */
    --text-primary: #1F2937;       /* 主要文字 */
    --text-secondary: #6B7280;     /* 次要文字 */
    --text-light: #9CA3AF;         /* 浅色文字 */

    /* 背景色 */
    --bg-gradient-start: #FFF7ED;  /* 渐变起点 */
    --bg-gradient-end: #FFEDD5;    /* 渐变终点 */
    --white: #FFFFFF;              /* 纯白 */

    /* 阴影 */
    --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
    --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.1);
    --shadow-lg: 0 10px 25px rgba(0, 0, 0, 0.15);
    --shadow-xl: 0 20px 40px rgba(234, 88, 12, 0.2);

    /* 圆角 */
    --radius-md: 12px;
    --radius-lg: 20px;
    --radius-xl: 28px;
}
```

### 使用场景
- **主色调**：按钮、图标、强调元素
- **浅橙色**：背景、徽章、状态提示
- **深橙色**：文字强调、深色主题
- **渐变色**：标题、图标、背景装饰

## 📐 布局适配

### 桌面版（>1024px）
- 左右两栏布局
- Hero区域内容并排
- 功能特性4列或3列网格
- 数据统计4列布局

### 平板版（768px - 1024px）
- 单栏布局
- Hero区域上下排列
- 功能特性2列网格
- 数据统计2列布局

### 手机版（<768px）
- 单栏布局
- 全宽设计
- 功能特性1列
- 数据统计1列
- 导航栏折叠
