## ❓ 关于宝塔

### 不用宝塔完全可以！

**为什么不用宝塔？**

1. ✅ 更轻量
   - 宝塔面板占用资源较多
   - 原生命令更高效

2. ✅ 更灵活
   - 完全掌控配置
   - 不受面板限制

3. ✅ 更安全
   - 少一个攻击面
   - 更精简的系统

4. ✅ 更免费
   - 宝塔高级功能收费
   - 原生工具全免费

**你需要用的工具（都是免费的）：**

| 工具 | 用途 | 替代宝塔的什么功能 |
|------|------|-------------------|
| SSH | 远程管理服务器 | 宝塔面板登录 |
| apt | 软件包管理 | 宝塔软件商店 |
| nano | 文本编辑 | 宝塔文件管理器 |
| mysql | 数据库管理 | 宝塔数据库管理 |
| systemctl | 服务管理 | 宝塔服务管理 |
| pm2 | 进程管理 | 宝塔进程管理 |
| nginx | 反向代理 | 宝塔网站管理 |

**这些工具比宝塔更轻量、更高效、更专业！**

## 📋 你需要填写的所有内容

### 服务器端（3个密码）

1. **MySQL数据库密码**
   - 位置：创建数据库时
   - 示例：`MyPassword123!`
   - 要求：包含大小写字母、数字、特殊字符

2. **JWT密钥**
   - 位置：`.env` 文件
   - 示例：`随便填一个长字符串abc123`
   - 要求：任意长字符串

3. **管理员密码**
   - 位置：`.env` 文件
   - 示例：`MyAdminPassword456`
   - 用途：登录管理后台

### Expo端（2个信息）

1. **注册邮箱**
   - 位置：https://expo.dev
   - 示例：`your_email@example.com`

2. **注册密码**
   - 位置：注册时设置
   - 示例：`YourPassword123`

**就这5个！其他都有默认值或自动生成！**
